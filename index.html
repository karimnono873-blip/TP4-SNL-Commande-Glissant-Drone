COMMANDE PAR MODES GLISSANTS

TP N°4 - Contrôle Robuste d'un Quadricoptère (SMC) à 6 DDL

1. Synthèse Analytique par Modes Glissants (Q2 & Q3)

L'objectif de la commande par modes glissants (Sliding Mode Control - SMC) est de forcer les trajectoires d'état à atteindre une surface prédéfinie \( S = 0 \), puis à glisser le long de celle-ci vers l'origine (erreur nulle).

A. Surface de Glissement et Dynamique d'Erreur

Définissons l'erreur de poursuite \( e_i = x_{id} - x_i \). La surface de glissement pour un système du second ordre est choisie de type Hurwitz :

\[ S_i = \dot{e}_i + \lambda_i e_i \quad \text{avec } \lambda_i > 0 \]

Prenons l'exemple du contrôle d'altitude \( z \) (où \( x_5 = z \), \( x_6 = \dot{z} \)). L'équation dynamique est :

\[ \dot{x}_6 = f_6(X) + g_6(X)U_z \] \[ f_6(X) = -\frac{1}{m}(k_{dz}x_6 + mg) \] \[ g_6(X) = \frac{\cos(\phi)\cos(\theta)}{m} \]

B. Loi de Commande et Preuve de Stabilité de Lyapunov (Q3)

Pour garantir la convergence vers la surface \( S_z = 0 \), nous utilisons une fonction candidate de Lyapunov définie positive : \( V = \frac{1}{2}S_z^2 \).

Pour assurer l'attractivité, sa dérivée doit être définie négative : \( \dot{V} = S_z \dot{S}_z < 0 \). On impose la condition stricte : \( \dot{S}_z = -K_z \text{sgn}(S_z) \).

\[ \dot{S}_z = \ddot{z}_d - \dot{x}_6 + \lambda_z \dot{e}_z \] \[ \dot{S}_z = \ddot{z}_d - (f_6(X) + g_6(X)U_z) + \lambda_z \dot{e}_z \]

En isolant \( U_z \), nous obtenons la loi de commande finale :

\[ U_z = \frac{m}{\cos(\phi)\cos(\theta)} \left( \ddot{z}_d - f_6(X) + \lambda_z \dot{e}_z + K_z \text{sgn}(S_z) \right) \]

function dX = quad_dynamics(t, X, pos_d, m, g, Ix, Iy, Iz, Ir, kdt, kdr, L, K)
    % Extraction des états X = [x, dx, y, dy, z, dz, phi, p, theta, q, psi, r]
    x1=X(1); x2=X(2); x3=X(3); x4=X(4); x5=X(5); x6=X(6);
    phi=X(7); p=X(8); th=X(9); q=X(10); psi=X(11); r=X(12);
    
    % Consignes actuelles
    ref = pos_d(t); xd = ref(1); yd = ref(2); zd = ref(3); psid = ref(4);
    
    % --- BOUCLE EXTERNE : Contrôle de Position (X, Y, Z) ---
    ez = zd - x5; evz = 0 - x6; Sz = evz + L(3)*ez;
    f6 = -(kdt*x6 + m*g)/m; g6 = cos(phi)*cos(th)/m;
    Uz = (1/g6) * (-f6 + L(3)*evz + K(3)*sign(Sz));
    
    % Suite de la boucle SMC (Ux, Uy) et Boucle Interne (Uphi, Uth, Upsi)...
end
